1. Butterfly主题

1.1 主题的安装

hexo最开始的主题是landscape,hexo有很多主题可选，最经典就是NextT了，非常简约，但是我也不喜欢这种风格。于是我就选了butterfly这款主题，此款主题由大佬Jerry开发，目前在持续更新中。每款主题都有对应的github地址，此款链接Butterfly,复制此链接。打开blog根目录下的themes,然后git bash here,执行git clone XXXX,(X出为主题链接)，接下来你会发现你的themes多了个butterfly。这时主题已经安装好了。

1.2 主题的配置

把根目录下的_config.yml中themes:landscape,改为butterfly就OK了，如果直接部署会出现extends includes/layout.pug block content include ./includes/mixins/post-ui.此时缺少pug & stylus renderer，还需要执行命令npm install hexo-renderer-pug hexo-renderer-stylus然后你就能用了。当然如果你想让你的博客与众不同，还需去butterfly下_config.yml中去更改相关配置。详细操作，请看butterfly主题官方文档

1.3 再次更改

由于将hexo部署在github pages上访问太慢，于是我将hexo重新部署在国内的码云上。具体操作和原来差不多。后期我将会把博客部署在个人的云服务器上。
